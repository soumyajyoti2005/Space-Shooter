# 🚀 Space Shooter

A classic space shooter game built with **Python** and **Pygame** where you defend against waves of UFOs!

## 🎮 Play Demo
👉 [Click here to play](https://soumyajyoti2005.github.io/Space-Shooter/build/web/)

---

## 🕹️ Controls
| Key | Action |
|-----|--------|
| ⬅️ Left Arrow | Move rocket left |
| ➡️ Right Arrow | Move rocket right |
| Space | Shoot bullet |

---

## 🎯 Gameplay
- Shoot down the **7 UFOs** before they reach your rocket
- UFOs move left and right and come closer each time they hit the boundary
- If any UFO crosses the boundary — **GAME OVER!**
- Each UFO you shoot gives you **+1 score**

---

## ⚙️ How to Run Locally

### 1. Clone the repo
```bash
git clone https://github.com/soumyajyoti2005/Space-Shooter.git
cd Space-Shooter
```

### 2. Install dependencies
```bash
pip install pygame
```

### 3. Run the game
```bash
python main.py
```
